Glossary on Java

  1. Variable :

    A named memory location used to store data. Example: int num = 5;

  2. Class:

    Blueprint for creating objects that defines its properties and behaviors.

    • Example:

      • public class Car {
      private String color;
      public void startEngine() {
      // Code to start the engine
      } }

  3. Method:

    A block of code that performs a specific task.

    • Example :

      • public int addNumbers(int a, int b)
      {
      return a + b;
      }

  4. Object:

    5. Instance of a class.

    • Example:

      • Car myCar = new Car();
      myCar.startEngine();

  5. Inheritance:

    6. Mechanism where a new class inherits properties and behaviors from another class.

    • example :

      • class SUV extends Car {
      // Additional SUV-specific properties and methods
      }

  6. Interface:

    7. A collection of abstract methods.

    • Example:

      • public interface Vehicle {
      void start();
      void stop();
      }

  7. Loop:

    8. Control flow statement that allows executing a block of code repeatedly.

    • example :

      • for (int i = 0; i < 5; i++)
      {
      // Code to be repeated
      }

  8. Polymorphism :

    9. Ability of an object to take many forms.

    • Example:

    class Animal {
    public void makeSound() {
    System.out.println("Some sound");
    } }
